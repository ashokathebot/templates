# Task Templates

Shared task templates for Ashoka The Bot 🧙 and Ashok.

## Pull Requests

| Template | Use |
|---|---|
| [`pull-request.md`](templates/pull-request.md) | Full PR — features, bugfixes, refactors |
| [`pull-request-quick.md`](templates/pull-request-quick.md) | Quick PR — typo fixes, one-liners |

## Coding

| Template | Use |
|---|---|
| [`bug-report.md`](templates/bug-report.md) | Consistent bug reports with severity |
| [`feature-request.md`](templates/feature-request.md) | Feature specs with acceptance criteria |
| [`code-review.md`](templates/code-review.md) | Review checklist and verdict |

## Tasks

| Template | Use |
|---|---|
| [`coding-task.md`](templates/coding-task.md) | Planner → coder → reviewer pipeline |
| [`research-task.md`](templates/research-task.md) | Spikes, comparisons, tech investigations |
| [`ops-task.md`](templates/ops-task.md) | Deployments, infra, config changes |

## Meta

| Template | Use |
|---|---|
| [`TEMPLATE.md`](templates/TEMPLATE.md) | Starting point for new templates |

## Usage

1. Copy the template to your task/project
2. Fill in the sections
3. Delete what doesn't apply
